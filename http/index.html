Introduction

This page provides a view of the stars from where you are, and a map of all the stars in the sky.

The tabs select one of a number of subpages;

Sky view
This is your window on to the stars in the sky; it can be zoomed, panned and rotated. An individual star can be selected. Most of the view is not affected by the observer position which is derived from the 'time', 'date', and 'location' settings. The top of the sky view has a green line; the left and right edges are blue, and the bottom is red. This window is shown on the sky map. The sky view exposes the controls - see below.
Sky map
The sky map is *all* of the stars above a certain magnitude, using a equirectangular cylindrical projection. The rectangular sky view windows is shown on the sky map; this usually does not appear as a rectangle (unless the field of view is small), but distorted. The top edge of the window is green (as in the sky view window), and the bottom edge of the window is red (etc). Clicking on the map will centre the sky view window on the that location. The sky map exposes the controls - see below.
Location
The location tab allows the setting of the location, by using a 3D model of the earth to select latitude and longitude. This is used for the observer position - which impacts the concept of 'up', and the direction of the compass, for example. It also permits the setting of the date and time to 'today'. More control is probably required...
Log
Many of the imporant user interface interactions are logged, so that the user can see (for example) which stars were selected in the past, or the direction the sky view was changed to. These events appear in the log.

Controls

The equatorial grid and/or azimuthal grid can be overlaid on top of the view (the azimuthal grid is affected by the observer position. The controls include a compass and an elevation setting: the compass can be swiped horizontally to move the view, setting 'up' to be vertically up from the observer position; the elevation can be swiped vertically to change the elevation of the view, again setting 'up' in a similar manner.

Equatorial grid

An 'equatorial' grid can be overlayed on top of the map, showing the equator, the 30th and 60th parallels, and great circles for every 30 degrees of longitude.

Azimuthal grid

An 'azimuthal' grid can be overlayed on top of the sky view and map, showing the grid as seen by the observer position (given by the location and time). This grid shows the great circles (for every 15 degrees of the compass, with north/south highlighted) that pass through the location of the observer, the horizon and circles of constant elevation (for every 30 degrees of elevation).
